# 🔐 SecureVault CLI

SecureVault CLI is a simple Java-based Command Line Interface (CLI) application that allows users to securely store and manage credentials locally.

---

## 📌 Overview
SecureVault CLI is a lightweight password manager built using Java. It helps users store website login details such as site name, username, and password using a menu-driven interface. The project focuses on Java fundamentals, file handling, and CLI application design.

---

## 🚀 Features
- ➕ Add new credentials (Site, Username, Password)
- 📄 View saved credentials
- 🔐 Basic authentication system (Master Password)
- 💾 Local file-based storage (storage.txt)
- 🖥️ Simple CLI menu interface

---

## 🛠 Tech Stack
- Java (Core)
- File Handling (I/O)
- Command Line Interface (CLI)
